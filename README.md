## Installation du backend API

Installer un serveur Mysql/MariaDB.

Lancer la commande :

```
make backend
```

Dans le dossier "api/utils/" et renommer le fichier ".env.production.example" en ".env.production" et editer le fichier.

## Installation du frontend

Lancer la commande :

```
make install
```
